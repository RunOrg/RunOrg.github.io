GET /db/{db}/forms/{id}

Returns a <form> representation of the form with identifier {id} in database {db}.

Example request

GET /db/0SNQc00211H/forms/0SNQe0032JZ
Content-Type: application/json

Example response

200 OK
Content-Type: application/json

{ "id" : "0SNQe0032JZ",
  "label" : "Personal information", 
  "owner" : "person",
  "custom" : null,
  "fields" : [ {
    "id" : "1",
    "label" : "Why did you join our group ?",
    "kind" : "text",
    "required" : true
  }, {
    "id" : "2", 
    "label" : "How often do you log in ?",
    "kind" : "single",
    "choices" : [
      "Every day",
      "Once a week",
      "Once a month",
      "Less than once a month" ],
    "required" : false
  } ],
  "access" : [ "admin", "fill" ],
  "audience" : {
    "admin": {},
    "fill": { groups: [ "0SNQe0032JZ" ] } } }

Errors

Returns 401 Unauthorized

  • if the provided token does not allow acting as {as}.

Returns 404 Not Found

  • if the database does not exist,
  • if form {id} does not exist in the database,
  • if person {as} does not have at least fill access to form {id}, to ensure absence equivalence.

Access restrictions

An access level of fill is required to view form information.
