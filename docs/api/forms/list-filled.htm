--- layout: docs-api title: List all filled instances of a form --- 
GET /db/{db}/forms/{id}/filled

Returns all the filled instances of form {id}, in an unspecified but consistent order, with pagination.

The response contains two fields:

Example request

GET /db/0SNQc00211H/forms/0SNQc00711H/filled?as=0Sd7003511H&limit=3&offset=213

Example response

200 OK
Content-Type: application/json 
   
{ "list" : [ { 
    "owner" : "0SNQe00311H",
    "data": { "color" : "Orange", "birth" : "1985-04-19" } 
  }, { 
    "owner" :  "0SNQe00311H",
    "data": { "birth" : "2013-10-08" }
  } ],
  "count": 215 }

Errors

Returns 401 Unauthorized

  • if the provided token does not authenticate the requester

Returns 403 Forbidden

  • if requester does not have admin access to the form.

Returns 404 Not Found

  • if database {db} does not exist
  • if form {id} does not exist in database {db}
  • if requester cannot view form {id}

Access restrictions

The admin access level is required to view all filled form instances.
