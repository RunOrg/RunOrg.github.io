--- layout: docs-api title: List all forms in the database --- 
GET /db/{db}/forms

Returns all the forms in the database, in an unspecified but consistent order, with pagination.

Only forms for which the viewer has fill access will be included in the list.

The response contains a single field:

The array contains forms, with two major differences:

  • Members custom and audience are not returned.
  • Member fields is replaced by the number of fields.

Example request

GET /db/0SNQc00211H/forms?limit=3&offset=16

Example response

200 OK
Content-Type: application/json 

{ "list" : [ 
  { "id" : "0SNQe00311H",
    "owner" : "person",
    "label" : "Why did you join our group ?", 
    "fields" : 2,
    "access" : ["fill"] },
  { "id" : "0SNQg00511H",
    "label" : null,
    "owner" : "person", 
    "fields" : 1,
    "pic" : ["admin","fill"] }

Errors

Returns 401 Unauthorized

  • if the provided token and viewer do not match.

Returns 404 Not Found

  • if the database does not exist

Access restrictions

Anyone can retrieve results from this API, though only forms that the viewer has fill access to will be returned.
