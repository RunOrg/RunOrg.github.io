GET /db/{db}/groups

Returns all the groups in the database, in an unspecified but consistent order, with pagination.

Only groups for which the viewer has view access will be included in the list.

Groups are returned in short format.

Example request

GET /db/0SNQc00211H/groups?limit=3&offset=16

Example response

200 OK
Content-Type: application/json 

{ "list" : [ 
  { "id" : "0SNQe00311H",
    "label" : null,
    "access": [ "list", "view" ],
    "count": 25 },
  { "id" : "0SNQg00511H",
    "label" : "Team members",
    "access" : [ "view" ]
    "count": null } ] }

Errors

Returns 401 Unauthorized

  • if the provided token and viewer do not match.

Returns 404 Not Found

  • if the database does not exist

Access restrictions

Anyone can retrieve results from this API, though only groups that the viewer has view access to will be returned.
