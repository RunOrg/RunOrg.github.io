type <group>

Returned by most API methods that involve groups. It is intended to provide data relevant for displaying the group: its label and count.

This object is viewer-dependent: its contents depend on the identity of the viewer.

id<id>

The identifier of the group in the database.

label<label>, null

An optional human-readable label. Not all groups have one, and it is not a mandatory property when creating a group.

access<access> array

The list of access levels available to the viewer.

count<int>, null

The number of group members (people currently in the group). This field is only available if the viewer has list access to the group.

{ "id" : "0Et9j0026rO",
  "label" : "Team members",
  "access" : [ "list", "view" ],
  "count": 16 }
