GET /db/{db}/groups/{id}

Returns all the people in the specified group, in arbitrary order.

Request

This is a GET request with the following parameters:

limit<int>

The maximum number of people to be returned.

offset<int>

The number of people to be skipped.

Response

This is a read-only request that always responds 200 OK and the following fields:

list<person> array

A list of short profiles.

count<int>

The total number of members in the group.

Example request

GET /db/0Et4X0016om/groups/0Et9j0026rO?limit=3&offset=213

Example response

200 OK 
Content-Type: application/json

Errors

Returns 401 Unauthorized

  • if the provided token does not allow acting as the viewer.

Returns 403 Forbidden

  • if the viewer does not have list access to the group.

Returns 404 Not Found

  • if the database does not exist,
  • if group {id} does not exist in the database.
  • if the viewer does not have at least view access to the group, to ensure absence equivalence.

Access restrictions

Viewing group members requires list access to that group.
