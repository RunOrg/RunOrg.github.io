class RunOrg.Person

This object contains basic information about a person, just enough to be able to display their identity :

id<id>

The identifier of the person in the database.

label<label>

A human-readable label. RunOrg will use the person's full name if available, and try to construct a label through another (undocumented) method otherwise.

gender"F", "M", null

The gender of the person.

static RunOrg.Person.ListList all people in the database
method RunOrg.Person.LoadFetch basic information for a person

For privacy purposes, the short information will never include contact information, such as a _full_ e-mail address or phone number. To ensure that this is the case, make sure you correctly classify any contact information as such.
