static RunOrg.Group.List

RunOrg.Group.List(limit, offset) reads limit groups from the database, starting at offset. The order is unspecified but consistent.

Only groups for which the viewer has view access will be included in the list.

Returns a promise that will resolve to a list of filled RunOrg.Group instances.

Example usage

RunOrg.Group.List(10, 0).then(function(list) {
  for (var i = 0; i < list.length; ++i) 
    console.log("Group %s is %s", list[i].id, list[i].label)    
})

Errors

Returns 401 Unauthorized

  • if the provided token and viewer do not match.

Returns 404 Not Found

  • if the database does not exist

Access restrictions

Anyone can retrieve results from this API, though only groups that the viewer has view access to will be returned.
