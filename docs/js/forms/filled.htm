class RunOrg.Form.Filled

A javascript representation of a filled instance of a form.

This object is viewer-dependent: its contents depend on the identity of the viewer.

Use RunOrg.Form.Fill(owner) to create a RunOrg.Form.Filled javascript object, which can then be used to read or write the filled form instance.

id<id>

The identifier of the form to which this filled instance belongs.

owner<id>

The identifier of the owner of this instance (e.g. a person's identifier).

data<dictionary>

The form data, maps field names to arbitrary JSON values. See reading and writing for more information about this field.

  "owner": "0Et9j0026rO",
  "data": { 
    "color": "Red",
    "birth": "1985-04-19" } }
