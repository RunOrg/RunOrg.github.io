static RunOrg.Form.List

RunOrg.Form.List(limit, offset) reads limit groups from the database, starting at offset. The order is unspecified but consistent.

Only groups for which the viewer has view access will be included in the list.

Returns a promise that will resolve to a list of filled RunOrg.Form instances, with two major differences:

  • Members custom and audience are not returned.
  • Member fields is replaced by the number of fields.

Example usage

RunOrg.Form.List(10, 0).then(function(list) {
  for (var i = 0; i < list.length; ++i) 
    console.log("Form %s has %d fields", list[i].id, list[i].fields)    
})

Errors

Returns 401 Unauthorized

  • if the provided token and viewer do not match.

Returns 404 Not Found

  • if the database does not exist

Access restrictions

Anyone can retrieve results from this API, though only forms that the viewer has fill access to will be returned.
